# CORRESPONDANCES.md — <NomDuProjet>

> Registre des échanges : emails, courriers, appels, réunions.
> Chaque échange porte un identifiant stable `C-XXXX`. Les pièces sont rangées dans `05_correspondances/`.
> Frontière : ici le registre logique des échanges ; les documents sources eux-mêmes vivent dans les dossiers, et les faits prouvés dans `PREUVES.md`.

## Format d'un échange

```
### C-XXXX — Objet de l'échange
- **Date** : YYYY-MM-DD
- **Canal** : email | courrier | téléphone | réunion
- **Interlocuteur** : qui.
- **Sens** : reçu | envoyé.
- **Résumé** : l'essentiel en une à deux phrases.
- **Document source** : 05_correspondances/YYYY-MM-DD-nom.pdf.
- **Action suivante** : ce que cet échange déclenche (avec échéance si besoin).
- **Lien** : P-XXXX, échéance, décision reliée.
```
