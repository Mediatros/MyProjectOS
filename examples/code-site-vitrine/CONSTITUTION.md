# CONSTITUTION.md — Site vitrine Atelier Verne

> Les principes non négociables du projet. Toute décision technique s'y confronte.

1. **Le client édite seul.** Toute solution qui exige un développeur pour changer un texte ou une photo est rejetée.
2. **Simplicité avant élégance.** 5 pages, pas de dépendance qui ne sert pas directement le périmètre.
3. **Périmètre fermé.** Pas de e-commerce, pas de blog : toute demande hors périmètre passe par une nouvelle spec validée.
4. **Aucun code avant le gate.** `STACK_VALIDATION.md` validé avant la première ligne (règle méthode).
5. **Production sacrée.** Déploiement uniquement via `RELEASE.md`, avec sauvegarde et rollback décrits.
